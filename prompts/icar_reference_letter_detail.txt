ICAR - Reference Letter Detail Prompt v2.0

Objective:
Reference Letter Detail Prompt is a versatile prompt designed to generate customized reference letters tailored to specific countries and their respective conventions. Its primary goal is to streamline the process of creating professional and culturally appropriate reference letters for employees in various international contexts.

Input:
Country: [Germany|Austria|Switzerland|UK|USA]
Type of reference letter: [Arbeitszeugnis|Referenzschreiben|Reference Letter]
Employee name: [Name]
Position: [Position]
Start date: [MM/YYYY]
End date: [MM/YYYY]
Evaluation:
Germany: Refer to the "Deutschland" table below for the appropriate evaluation criteria and ratings.
Austria: Refer to the "Österreich" table below for the appropriate evaluation criteria and ratings.
Switzerland: Refer to the "Schweiz" table below for the appropriate evaluation criteria and ratings.
UK: Refer to the "UK" table below for the appropriate evaluation criteria and ratings.
USA: Refer to the "USA" table below for the appropriate evaluation criteria and ratings.

Output:
Generated reference letter tailored to the selected country and type of reference letter, incorporating the evaluation criteria and ratings from the corresponding country-specific table and following the structure provided in the country-specific templates.

Usage Instructions:
Specify the country for which you need the reference letter.
Select the type of reference letter you want to create.
Provide the employee's name, position in the company, start date, and end date.
Evaluate the employee's work performance based on the relevant evaluation criteria and ratings for the selected country, as specified in the corresponding table below.
Review the generated reference letter for completeness and accuracy before finalizing it.

General Application Procedure:
Use the country-specific templates, terminologies, and evaluation criteria listed in the respective tables and templates below.
Provide all required information accurately to ensure that the reference letter meets the standards of the selected country.
Review the generated reference letter for completeness and accuracy before finalizing it.

Additional Information:
For reference letters in different languages, specific questions and language conventions will be adapted to the selected country.
Provide accurate and complete information to generate a customized reference letter.

Explanation of Ratings:
Excellent (Sehr gut): This rating is used to highlight outstanding performance or skills of the employee. It should only be awarded when the employee significantly exceeds expectations and has delivered exceptional results.
Good (Gut): This rating is given for above-average performance or skills of the employee. It indicates that the employee has met or exceeded expectations and has overall delivered strong performance.
Satisfactory (Befriedigend): This rating is awarded when the employee has performed tasks or demonstrated skills satisfactorily. It indicates that the employee has met the basic requirements, but there may be room for improvement.
Sufficient (Ausreichend/Genügend): This rating is given when the employee has met the minimum requirements but has not delivered outstanding performance. It indicates that the employee has completed the required tasks, but there may be potential for improvement.
Deficient (Mangelhaft/Nicht genügend): This rating is awarded when the employee has not satisfactorily fulfilled the required tasks or skills and exhibits significant deficiencies. It indicates that the employee has had difficulties meeting the requirements and may urgently need support or training.
Insufficient (Ungenügend): This rating is given when the employee has not met the minimum requirements and the performance is not acceptable. It indicates that the employee has significant difficulties in fulfilling the job requirements and may not be suitable for the position.
