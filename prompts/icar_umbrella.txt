Objective: ICAR Umbrella Prompt is a high-level prompt designed to generate customized cover letters, resumes, and reference letters tailored to specific job postings and country-specific conventions. Its primary goal is to provide an overview of the process of creating professional application and reference documents that highlight the candidate's strengths, experiences, and qualifications in a streamlined and efficient manner.
Input:
Candidate's resume, articles, certificates, and other relevant artifacts
Job posting or description or reference letter requirements
Candidate's preferred language for the output
Output:
Generated cover letter and resume tailored to the job posting, incorporating the candidate's qualifications, experiences, and strengths (see ICAR_COVER_LETTER_DETAIL_PROMPT-v1.2 for detailed instructions)
Generated reference letter tailored to the selected country and type of reference letter incorporating the evaluation criteria and ratings from the corresponding country-specific table (see ICAR_REFERENCE_LETTER_DETAIL_PROMPT-v2.0 for detailed instructions)
Candidate Evaluation Matrix, Potential Role Alignment Table, Assessment Summary, and Hiring Rationale sections
Usage Instructions:
Provide the candidate's resume, articles, certificates, and other relevant artifacts.
Specify the job posting or description or reference letter requirements for which the application documents are being created.
Indicate the candidate's preferred language for the output.
Review the generated cover letter, resume, and reference letter for completeness, accuracy, and consistency with the provided artifacts, job posting, and country-specific conventions.

General Application Procedure:
Analyze the provided artifacts to identify relevant themes, skills, and experiences.
Assess the candidate's qualifications and evaluate their expertise based on the artifacts.
Adapt the application focus to the job posting and create a tailored cover letter and resume.
Create a tailored reference letter based on country-specific requirements and evaluation criteria.
Incorporate feedback mechanisms to ensure the quality and relevance of the generated documents.
Address data privacy and security concerns, ensuring compliance with regulations and best practices.
Feedback Loop for Detail Integration:
Conduct a detailed analysis of the candidate's resume, articles, certificates, and other relevant artifacts.
Identify themes, skills, and experiences from the artifacts.
Verify the candidate's personal information and correct any errors.
Assess qualifications and create a KPI metric for evaluation.
Compare the analysis with job posting requirements and country-specific reference letter standards.
Incorporate feedback from employers or stakeholders.
Summarize the assessment with a recommendation and match percentage.
Adapt the cover letter, resume, and reference letter according to feedback.
Review for completeness, accuracy, and alignment with job requirements and reference letter conventions.
Continuously iterate to improve the final output.
Meta-Feedback Loop: All components of ICAR Umbrella, Cover Letter Detail Prompt, Reference Letter Detail Prompt, and the Knowledge Base (if available) undergo a meta-feedback loop before finalizing the response, repeating iterations until the objective of the request is met with the highest precision and quality.

This continuous exchange ensures the general logic and detailed logic of ICAR are aligned, allowing for iterative refinement based on feedback.
